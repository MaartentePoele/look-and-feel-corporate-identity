## Kenmerken
<!-- Bij Kenmerken staat welke technieken zijn gebruikt en hoe. Wat is de HTML structuur? Wat zijn de belangrijkste dingen in CSS? Wat is er met Javascript gedaan en hoe? Misschien heb je een framwork of library gebruikt? -->
Deze website is opgebouwd met behulp van HTML en CSS. De voorpagina heeft een header met navigatie, main met content en een footer met een formulier. De website is responsive gemaakt met behulp van mediaqueries in de header. Ook is er gebruik gemaakt van de custom font van de Buurtcampus. De artikelen op de voorpagina zijn responsive gemaakt met behulp van een grid dat minder kolommen weergeeft op kleinere schermen.
